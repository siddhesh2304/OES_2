----------------------------------------------Installation-----------------------------------------  
1.Unzip examsystem.zip (executable folder has Home.jar which doesn’t need any libraries and it's independent ,you can double click to execute it   ).
To edit a project ,you need to compile the project again.

2.copy ucanacess_lib to C:\Program Files\Java\jdk1.8.0_231 (or your jdk installation )

3. Now we'll register environment variables  (reference video for  environment variables https://www.youtube.com/watch?v=8R4e2HxFSsk )

    A. Open “Control Panel\All Control Panel Items\System”
   B. Click advanced system settings
   C. click environment variables
   D.  click on new button in “ user variables for (your_pc_name)
   E.  Enter variable name as CLASSPATH
   F. links to all the lib
      C:\Program Files\Java\jdk1.8.0_231\lib\UCanAccess-5.0.0-bin\ucanaccess-5.0.0.jar
      C:\Program Files\Java\jdk1.8.0_231\lib\UCanAccess-5.0.0-bin\lib\commons-lang3-3.8.1.jar
      C:\Program Files\Java\jdk1.8.0_231\lib\UCanAccess-5.0.0-bin\lib\commons-logging-1.2.jar
      C:\Program Files\Java\jdk1.8.0_231\lib\UCanAccess-5.0.0-bin\lib\hsqldb-2.5.0.jar
      C:\Program Files\Java\jdk1.8.0_231\lib\UCanAccess-5.0.0-bin\lib\jackcess-3.0.1.jar

4.Now copy all source code files in
        C:\Program Files\Java\jdk1.8.0_231\bin
5. compile all java files you copied with javac
6.run project using command “java Home”

Note database path is defined in ExamData.java file
